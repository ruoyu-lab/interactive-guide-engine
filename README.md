# Interactive Guide Engine

一个用于构建交互式教程、功能引导、新手引导和任务流程的引擎。

它不是普通的“下一步教程弹窗”，而是一个可以：

- 显示提示
- 等待用户完成真实操作
- 判断操作是否正确
- 自动进入下一步
- 保存教程进度
- 支持跳过、回退、重试和完成状态

的交互式引导框架。

## 适用场景

Interactive Guide Engine 适合用于：

- Web 应用新手引导
- 游戏教程
- 节点编辑器教学
- 后台系统功能引导
- 交易工具操作教学
- 可视化编辑器 onboarding
- 复杂业务流程引导
- 步骤式任务系统

例如：

```txt
显示提示 → 等待用户点击按钮 → 检查状态是否正确 → 自动进入下一步# interactive-guide-engine
